# Make CLI Executable from Anywhere

## 概要
現在のabm_checkは`python -m abm_check`という形式で実行する必要があり、プロジェクトディレクトリに移動する必要があります。このプロポーザルでは、`abm_check`コマンドが任意のディレクトリから実行可能になるように変更します。

## 現状の問題点
- `python -m abm_check`形式での実行にはプロジェクトディレクトリへの移動が必要
- ユーザーエクスペリエンスが低下
- 自動実行スクリプトでの使用が不便

## 解決策
- インストール時に`abm_check`コマンドがシステムのPATHに追加されるように設定を確認/強化
- インストール手順を明確化し、ユーザーが`abm_check`コマンドをどこからでも実行できるようにする

## 期待される影響
- ユーザーの利便性が向上
- 任意のディレクトリからCLIが実行可能に
- cron等の自動実行にも適した形に
